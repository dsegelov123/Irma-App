# Irma App - UI Design System & Brand Guidelines

This document serves as the single source of truth for the typography, font weights, size hierarchy, and brand color palette of the **Irma** application.

---

## 1. Typography

The application uses the **Urbanist** font family, a simple, geometric, and elegant sans-serif typeface.

### Font Weights
- **Medium**: `500` (Default Body Text)
- **Semi-Bold**: `600` (Sub-headers & Accents)
- **Bold**: `700` (Display & Labels)

---

## 2. Text Style Hierarchy

The design system maps text sizes into specific layout components:

### Display & Hero Text (Weight: `700`)
- **Display lg**: `180px`
- **Display md**: `128px`
- **Display sm**: `96px`

### Headings (Weight: `700` / `600`)
- **Heading 2xl**: `72px`

### Body Paragraphs (Weight: `500`)
- **Paragraph 2xl**: `24px`
- **Paragraph xl**: `20px`
- **Paragraph lg**: `18px`

- **Paragraph sm**: `14px`

### Labels & Action Text (Weight: `700`)
- **Label 2xl**: `20px`
- **Label xl**: `18px`
- **Label lg**: `16px`

- **Label sm**: `12px`
- **Label xs**: `10px`

## 4. Buttons & Controls

The buttons and input controls in the Irma design system are categorized by their styling, sizing, corner radius, and padding behaviors:

### Primary & Secondary Buttons (`button-primary` & `button-secondary`)
- **Primary Button**: Fills are **Warm Earthy Brown** (`#4B3425`). Outlined variant has a transparent fill with a brown border.
- **Secondary Button**: Fills are **Light Gray** (`#F7F3F2` / `rgb(247, 243, 242)`). Outlined variant has a transparent fill with a light gray border.
- **Sizing & Padding (Pill Shape / Radius: `1000`)**:
  - **Large (`lg`)**: Padding: Horizontal `32px`, Vertical `16px`.
  - **Medium (`md`)**: Padding: Horizontal `24px`, Vertical `16px`.
  - **Small (`sm`)**: Padding: Horizontal `20px`, Vertical `8px`.
  - **Extra Small (`xs`)**: Padding: Horizontal `20px`, Vertical `8px`.

### Specialized Interactive Buttons
- **Date Range Button (`button-date-range`)**:
  - **Selected**: Fill: Warm Earthy Brown (`#4B3425`). Padding: `10px` on all sides. Radius: Circular/Pill.
  - **Default**: Fill: Light Gray (`#F7F3F2`). Padding: `10px` on all sides. Radius: Circular/Pill.
- **AI Model Radio (`conversation-ai-model-radio`)**:
  - **Selected**: Fill: Sage Green (`#9BB068`). Padding: `16px` on all sides. Radius: Circular/Pill.
  - **Default**: Fill: White (`#FFFFFF`). Padding: `16px` on all sides. Radius: Circular/Pill.
- **Toggle Button (`toggle-button`)**:
  - **True**: Fill: Sage Green (`#9BB068`). Padding: `0px`. Radius: Circular/Pill.
  - **False**: Fill: Light Tan/Gray (`#E8DDD9`). Padding: `0px`. Radius: Circular/Pill.
- **Icon Button (`button-icon`)**:
  - **Sizes (`sm` to `2xl`)**: Fill: Empathy Orange (`#FE814B` / `rgb(254, 128, 75)`). Padding: `0px`. Radius: Pill.
- **Conversation Style Pill (`conversation-style-pill`)**:
  - **Selected**: Fill: Very Light Sage/Green tint (`#EDEBFF` / `rgb(242, 244, 234)`). Padding: `16px` on all sides. Radius: Pill.
  - **Default**: Fill: White (`#FFFFFF`). Padding: `16px` on all sides. Radius: Pill.

## 5. Implementation Guidelines (Flutter/Dart)

To implement this design system in Flutter, we will map these tokens directly into our `ThemeData` and custom widget properties:

```dart
import 'package:flutter/material.dart';

class IrmaTheme {
  // Brand Colors
  static const Color darkEspresso = Color(0xFF1F160F);
  static const Color earthyBrown = Color(0xFF4B3425);
  static const Color sageGreen = Color(0xFF9BB068);
  static const Color pureWhite = Color(0xFFFFFFFF);
  static const Color lightGray = Color(0xFFF7F3F2);
  static const Color empathyOrange = Color(0xFFFE814B);
  static const Color lightTan = Color(0xFFE8DDD9);
  static const Color lightSageTint = Color(0xFFEDEBFF);

  static ThemeData get lightTheme {
    return ThemeData(
      useMaterial3: true,
      fontFamily: 'Urbanist',
      scaffoldBackgroundColor: pureWhite,
      colorScheme: const ColorScheme.light(
        primary: sageGreen,
        secondary: earthyBrown,
        surface: pureWhite,
        onSurface: darkEspresso,
      ),
      textTheme: const TextTheme(
        displayLarge: TextStyle(fontSize: 180, fontWeight: FontWeight.w700, color: earthyBrown),
        displayMedium: TextStyle(fontSize: 128, fontWeight: FontWeight.w700, color: earthyBrown),
        displaySmall: TextStyle(fontSize: 96, fontWeight: FontWeight.w700, color: earthyBrown),
        headlineLarge: TextStyle(fontSize: 72, fontWeight: FontWeight.w700, color: earthyBrown),
        bodyLarge: TextStyle(fontSize: 16, fontWeight: FontWeight.w500, color: darkEspresso),
        labelLarge: TextStyle(fontSize: 14, fontWeight: FontWeight.w700, color: darkEspresso),
      ),
    );
  }
}
```

## 6. Cards & List Items

Cards in the Irma design system use structured padding, rounded corners, and soft, natural background fills to group information visually:

### Large Layout Cards (`notification-card`, `settings-card`, `faq-card`)
- **Corner Radius**: `32px`
- **Padding**: `24px` on all sides (top, bottom, left, right).
- **Fills**: Default transparent background (takes on container surface color, e.g., White `#FFFFFF` in light mode).

### Standard Action Cards (`mindful-journal-card`, `assessment-medication-card`)
- **Corner Radius**: `32px`
- **Padding**: `16px` on all sides.
- **Fills**: Solid White (`#FFFFFF`).

### Statistic & Content Cards (`chatbot-conversation-card`, `resources-article-card`, `sleep-statistics-card`)
- **Corner Radius**: `24px`
- **Padding**: `16px` on all sides.
- **Fills**: Solid White (`#FFFFFF`) or Light Gray (`#F7F3F2` for upgrade prompts).

### Score History & Log List Cards (`score-history-card`)
- **Corner Radius**: `24px`
- **Padding**: `12px` on all sides.
- **Fills**: Light Gray (`#F7F3F2`).

### Error & Verification Bubbles (`onboarding-otp-error`, `password-creation-error`)
- **Corner Radius**: Pill-shaped (fully rounded).
- **Padding**: `6px` to `8px` on all sides.
- **Fills**: Light Coral/Orange tint (`#FFD2C2` / `rgb(255, 210, 194)`).

### Input Pickers (`sleep-hour-picker`, `faq-live-chat-picker`)
- **Sleep Hour Picker**: Radius: `48px`. Fill: Sage Green (`#9BB068`).
- **Live Chat Picker**: Radius: Pill-shaped. Fill: Light Sage Green (`#B4C48D`).

  - **Extra Large (`xl`)**: Padding: Horizontal `32px`, Vertical `20px`.

### Plain Labels (`label-plain` component)
- **Corner Radius**: `0px` (no background container).
- **Padding**: `0px` on all sides.
- **Sizes**: `sm`, `md`, `lg`.

## 8. Forms & Input Fields

Form inputs in the Irma design system are built around a flexible, padded component set supporting labels and icon placements:

### Master Input Form (`input-form-master`)
- **Corner Radius**: `32px`
- **Padding**: `24px` on all sides.
- **Border**: `1px` stroke width, colored in **Sage Green / Green 50** (`#9BB068`).
- **Layout (Auto-layout Row)**:
  - **Direction**: Horizontal/Row
  - **Gap**: `24px` (spacing between text and prefix/suffix icons)
  - **Alignment**: Align main axis to `start`, cross axis to `center`.
- **Variants Supported**:
  - **Label Configuration**: `label-true` or `label-false`
  - **Icon Positions**:
    - `icon-start` (prefix icon)
    - `icon-end` (suffix/action icon)
    - `icon-start---end` (both prefix and suffix icons present)
    - `icon-none` (text-only field)
